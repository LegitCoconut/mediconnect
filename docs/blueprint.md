# **App Name**: MediConnect Pro

## Core Features:

- Hospital Registration: Allow hospitals to register and create a detailed profile, including location, services, and operating hours.
- Doctor Management: Enable hospitals to manage doctor profiles, including adding, editing, and deactivating doctors.
- Appointment Management: Facilitate appointment scheduling and management with status updates (pending, confirmed, completed, cancelled) and filtering options.
- Prescription Generation: Allow doctors to create and manage prescriptions for patients, including medication details, dosage, and instructions.
- Admin Hospital Verification: Enable administrators to verify and manage hospital accounts, ensuring only approved hospitals can access the platform.
- Mobile API: Provide API endpoints for mobile applications, including hospital listing, doctor search, appointment booking, and prescription retrieval.

## Style Guidelines:

- Primary color: Teal (#26A69A) for a clean and professional feel, reminiscent of healthcare environments.
- Background color: Dark gray (#212121) to create a modern dark theme that is easy on the eyes, with a desaturated tone.
- Accent color: Blue (#64B5F6) to highlight key interactive elements and call-to-actions, contrasting with the teal and dark gray.
- Headline font: 'Poppins' sans-serif for headlines and other prominent text elements, to provide a modern and precise look.
- Body font: 'Inter' sans-serif for body text, which complements Poppins and is very readable in digital environments.
- Use Lucide React icons throughout the application for a consistent and modern visual language.
- Implement a glassmorphism design for cards, incorporating backdrop blur and subtle transparency to create a sophisticated and engaging UI.
